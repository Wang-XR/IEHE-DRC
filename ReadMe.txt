Read Me
This repository provides the data and code for a demonstration of the paper "Real-time food insecurity assessment with deep ensemble learning: a solution without primary survey data".
Data
The Food_dataset.csv file contains data on various features, including the prevalence of people with insufficient food consumption (PIFC), the prevalence of people using crisis or above-crisis food-based coping (PCFC), and other related indicators, for subnational administrative units across different countries/regions.
The first_level_admin_area_codes.csv file contains the codes for countries/regions and their subnational administrative units.
The population.csv file contains the population data for each subnational administrative unit.
Run the code
The code requires Python 3.10.
To run the code create a pip virtual environment and install the required packages via:
`pip install -r requirements.txt`
